Getting started with the Kinect and Ubuntu

1: Download and build OpenNI

https://github.com/OpenNI/OpenNI

(Note: If you get an error when loading Microsoft.Windows.Forms or something similar when building, you need to do: sudo apt-get install mono-complete)

2: Download and build SensorKinect

https://github.com/avin2/SensorKinect

3: Download and build NITE http://www.openni.org/downloadfiles/opennimodules/openni-compliant-middleware-binaries/latest-unstable/113-primesense-nite-unstable-build-for-ubuntu-10-10-x64-64-bit-v1-3-1
