LimeJS

A framework for making HTML5 games.

Property Investment Australia

Elevator Pitch

Easy framework for web devs to build 2D games quickly for browsers and touchscreens.

Features

  • Director, Scenes and Entities (LimeJS calls these “Nodes”)
  • Box2d physics engine
  • Closure API
  • Sprite Sheets
  • Support for DOM, Canvas and WebGL rendering
  • CSS3 optimization - great for iOS hardware acceleration
